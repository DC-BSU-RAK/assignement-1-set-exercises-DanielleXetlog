HTML Reference

HTML TAGS

Replace the "()" to "<>" so it will activate.

html

- The root element for the HTML document and all tags are kept in it.


    (!DOCTYPE html)
    (html)
    (head)
        (title) Page Title (/title)
    (/head)
    (body)
        (h1) This is a Heading (/h1)
        (p) This is a paragraph. (/p)
    (/body)
    (/html)

head

- This tag holds the information of the page, such as the title, and it doesn't show up on the screen.


    (head)
        (title)My Webpage(/title)
        (meta charset="UTF-8")
        (link rel="stylesheet" href="styles.css")
    (/head)

body

- This holds everything you see on the webpage.


    (body)
        (h2)Welcome(/h2)
        (p)This is the visible content.(/p)
        (div)Extra section(/div)
    (/body)

h1

- This makes the text big, being an important title. There is also h2, h3, h4, h5, h6 for smaller text.


    (h1)Biggest Heading(/h1)
    (h2)Second Level(/h2)
    (h3)Third Level(/h3)
    (h4)Fourth Level(/h4)
    (h5)Fifth Level(/h5)
    (h6)Smallest Heading(/h6)

p

- This makes a block of text like a paragraph.


    (p)This is a paragraph of text.(/p)
    (p)Another paragraph goes here.(/p)

div

- This is a box where there groups inside like styling.


    (div)
        (h2)Section Title(/h2)
        (p)Content inside a div.(/p)
    (/div)

ul

- This makes a bullet point list.


    (ul)
        (li)Item 1(/li)
        (li)Item 2(/li)
        (li)Item 3(/li)
    (/ul)

title

- This is the name you will see on your browser tab.


    (head)
        (title)My Webpage(/title)
    (/head)

a

- Makes the text into a clickable link to another page of the website.


    (a href="https://example.com")Click here(/a)

li

- Adds an item to the list.


    (ul)
        (li)First item(/li)
        (li)Second item(/li)
    (/ul)
