CSS PROPERTIES

color

This makes your text into the color you've selected.


    .text {
        
        color: #e74c3c;         
        background-color: #f8f9fa;
        font-size: 20px;          
        margin: 15px;             
        padding: 10px;            
        border: 1px solid #ddd;   
        width: 250px;             
        text-align: center;       
          }

background-color

- This makes the background color the color you've selected.


    .text {
        color: #e74c3c;         
        background-color: #f8f9fa;
        font-size: 20px;          
        margin: 15px;             
        padding: 10px;            
        border: 1px solid #ddd;   
        width: 250px;             
        text-align: center;       
          }

font-size

- This makes your text big or small on how many pixels you put in.


    .text {
        color: #e74c3c;         
        background-color: #f8f9fa;
        font-size: 20px;         
        margin: 15px;             
        padding: 10px;            
        border: 1px solid #ddd;   
        width: 250px;             
        text-align: center;       
          }

margin

- This adds space on the outside of your element.


    .text {
        color: #e74c3c;         
        background-color: #f8f9fa;
        font-size: 20px;         
        margin: 15px;           
        padding: 10px;            
        border: 1px solid #ddd;   
        width: 250px;             
        text-align: center;       
      }

padding

- This adds space inside the element between your content and border.


    .text {
        color: #e74c3c;         
        background-color: #f8f9fa;
        font-size: 20px;         
        margin: 15px;           
        padding: 10px;            
        border: 1px solid #ddd;   
        width: 250px;             
        text-align: center;       
      }

font-family

- This is an element that sets the font style.


    .text {
        color: #e74c3c;         
        background-color: #f8f9fa;
        font-size: 20px;         
        margin: 15px;           
        padding: 10px;            
        border: 1px solid #ddd; 
        font-family: Arial, Helvetica, sans-serif; 
        width: 250px;             
        text-align: center;       
      }

width

- This sets how long will your element be.


    .text {
        color: #e74c3c;         
        background-color: #f8f9fa;
        font-size: 20px;         
        margin: 15px;           
        padding: 10px;           
        border: 1px solid #ddd;   
        width: 250px;             
        text-align: center;       
      }

text-align

- This makes your text align inside the element.


    .text {
        color: #e74c3c;         
        background-color: #f8f9fa;
        font-size: 20px;         
        margin: 15px;           
        padding: 10px;           
        border: 1px solid #ddd;   
        width: 250px;           
        text-align: center;       
      }
